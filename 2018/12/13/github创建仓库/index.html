github初使用

发表于 2018-12-13 | 分类于 github

很久没有看我的GitHub了，只提交过一次，上面有一个交作业的仓库，看了下，还是太菜了，于是删除想重新操作一下GitHub，记录一下上传本地项目到Github的步骤。

  1. 打开GitHub：GitHub传送门
  2. 点击You profile，进入仓库管理
  3. 新建仓库，命名Demo
  4. GitHub是基于git的代码托管，所以你需要安装git—>git安装
  5. 新建Demo文件夹，你的代码扔进去，右键点击Demo文件夹根目录，点击“Git Bash Here”，打开git命令行。
  6. -1.在命令行中，输入“git init”，使Demo文件夹加入git管理；
    -2.输入“git add .”（不要漏了“.”），将Demo文件夹全部内容添加到git。
    -3.输入“git commit -m “first commit””（“git commit -m “提交信息””
    ）
    -4.输入“git remote add origin https://github.com/superxzl/Demo.git”（git remote add origin 你自己的https地址），连接你的guthub仓库。
    -5.输入“git push -u origin master”，上传项目到Github。这里会要求输入Github的账号密码，按要求输入就可以。
  7. 记录一下github创建仓库操作鸭！
